开始使用Hexo

2025-07-26 分类: 教程

Hexo是一个快速、简洁且高效的博客框架。使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。本文将介绍如何开始使用Hexo搭建自己的博客。

安装前提

安装Hexo相当简单，只需要先安装下列应用程序：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

安装Hexo

所有必备的应用程序安装完成后，即可使用npm安装Hexo。

$ npm install -g hexo-cli

建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init 
$ cd 
$ npm install

新建完成后，指定文件夹的目录如下：

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

配置

你可以在 _config.yml 中修改大部分的配置。

网站信息

# Site
title: 我的博客
subtitle: '副标题'
description: '网站描述'
keywords: 关键词
author: 作者名
language: zh-CN
timezonee: 'Asia/Shanghai'
